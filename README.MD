## **문제 상황:**

현대 사회에서 혼자 사는 1인 가구가 급격히 늘어나면서, 많은 사람들이 식사에서 **사회적 고립**을 경험하고 있다.

특히 족발, 치킨, 고기와 같이 **혼자 먹기에는 양이 많거나 세트로만 판매되는 음식**들은 혼자서는 즐기기 어렵다.

이로 인해 1인 가구는 먹고 싶은 음식을 포기하거나, 억지로 혼자 많은 양을 먹어야 하는 불편함을 겪는다.

또한, 일부 식당은 기본적으로 **2인 이상 주문**을 요구하거나 **혼밥을 꺼리는 분위기**가 강해, 혼자 사는 사람들이 이용하기 더욱 어렵다.

이러한 상황은 단순한 불편을 넘어, ‘밥조차 혼자 해결해야 하는 사회적 고립감’을 강화시키는 요인으로 작용한다.

따라서 식사를 함께할 사람을 찾고, 같은 음식을 나누어 즐길 수 있도록 연결해주는 서비스가 필요하다.

**주 서비스 내용**: 

혼자 먹기 부담스러운 음식들(족발, 치킨, 세트메뉴, 혼밥 안되는 집, 고기)을 같이 먹을 수 있는 사람들

`타겟층:` 혼자 사는 2030, 자주 혼밥하는 사람들

단기적인 사회적 고립을 해결

- 주요 기능:
    - **같은 시간·같은 메뉴를 원하는 근처 사람들**을 1~4명 소규모로 빠르게 매칭
    - create, join
    - 메뉴, 거리, 인원 별 필터링
    1. 빠른 매칭 - 혼자 먹기 어려운 메뉴를 원하는 시간에 바로 매칭할 수 있는 기능
    2. 위치 기반 매칭 - 사용자의 현재 위치를 기준으로 가까운 이웃과 연결되어 편리하게 만남 가능
    3. 소규모 그룹 형성 - 1~4명이 함께하는 소규모 그룹 위주로 부담 없이 식사 자리를 만들 수 있음
    4. 실시간 알림 - 새로운 매칭이나 그룹 활동 소식을 즉시 받아볼 수 있어 놓치지 않고 참여 가능
    5. 그룹 찾기 & 만들기 - 원하는 그룹에 참여하거나 직접 새 그룹을 만들어 모임을 주도할 수 있음
    6. 지금 모집 중인 그룹 보기 - 현재 참여 가능한 그룹들을 카드 형태로 확인하고, 시간·장소·인원 정보를 확인해 바로 참여할 수 있음
